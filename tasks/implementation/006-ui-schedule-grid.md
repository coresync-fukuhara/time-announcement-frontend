# 006: 曜日タブ・時刻グリッド画面の実装

- ステータス: 未着手
- 関連文書: [概要設計書](../../docs/schedule-ui-overview-design.md) 5章、[要件不足事項一覧](../../docs/schedule-ui-open-questions.md) No.1・No.2・No.6・No.10

## 概要

曜日タブ(`holiday` 含む全 8 タブ)+ 時刻グリッドの 1 画面 UI を実装する。

## 完了条件

- [ ] 曜日タブ(月〜日 + holiday)の切り替えができる
- [ ] 時間行の追加(`[+ 時間を追加]`、0〜23 時から未使用の時間を選択)ができる
- [ ] 時間行の削除ができる
- [ ] 各時間行に 0/5/10/…/55 の 5 分刻みボタン(12 個)があり、クリックで ON/OFF をトグルする
- [ ] 曜日間コピー機能(ある曜日の設定を他曜日へコピー)がある
- [ ] 未保存の変更がある場合にインジケーターが表示され、保存後に消える
- [ ] `[保存]` ボタン押下で `PUT /api/schedules` に送信する(トグル即時保存はしない)
- [ ] 保存前バリデーションエラー時は画面遷移しない
- [ ] `minute_settings` は編集対象外(UI に出さない)
- [ ] コンポーネントテスト(Vitest + React Testing Library)が green
