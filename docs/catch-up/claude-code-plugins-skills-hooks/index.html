このリポジトリ(Next.js + React + TypeScript、Vitest/RTL/Playwright/MSW/Ajv、 devcontainer + Claude Code)に有用そうな Plugin・Skill・Hook を、 公式または GitHub star 数が多く信頼できるものに絞って調査しました。 調査結果は docs/catch-up/claude-code-plugins-skills-hooks/research.md を元にしています。

この教材の使い方

当初は特定のツールの導入を決定するものではなく、カタログとして候補を並べるだけの教材だった。 その後ユーザーの指示により、公式Anthropicプラグイン7つ(frontend-design・feature-dev・ pr-review-toolkit・commit-commands・claude-md-management・hookify・security-guidance)を 実際に apm install で apm.yml に追加した(詳細は 05-comparison-summary.html の導入状況列)。 残りの候補は引き続き判断材料として掲載している。前回の agent-tool-management/ が 「管理ツールそのもの」を比較したのに対し、本教材は「その管理ツールに乗せる具体的な中身 (プラグイン・スキル・フック)」を扱います。
