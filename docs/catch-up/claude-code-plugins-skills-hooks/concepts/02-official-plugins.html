Anthropic公式マーケットプレイスの厳選プラグイン

マーケットプレイス: github.com/anthropics/claude-plugins-official · 2026年5月22日公開・255プラグイン掲載(2026-07時点)。Claude Code起動時に自動で参照可能。 ここで紹介する9つはいずれも author: Anthropic(一次配布)。

説明

255プラグインの大半はSaaS連携(AWS・Salesforce・Datadog等)で、 家庭内LAN・単一利用者運用の本プロジェクトには不要。 その中から「TypeScript/Next.js開発」「コード品質」「プロジェクト運用」に関連する 9つを抜粋した。

開発・コード品質系

プラグイン概要本プロジェクトでの用途
typescript-lsp TypeScript/JavaScript 言語サーバーによるコード解析強化 Next.js実装フェーズで型エラー・参照関係をエージェントが正確に把握できるようになる
frontend-design 高品質・独自性のあるフロントエンドUI実装を支援 曜日タブ×時刻グリッドのUI実装で、汎用的すぎないデザインを狙う際に有用
feature-dev コードベース探索・アーキテクチャ設計・品質レビューの専門エージェントによる 機能開発ワークフロー schedule-store.ts や API Route Handler など、まとまった機能を1から実装する際に使える
pr-review-toolkit コメント・テスト・エラー処理・型設計・簡潔化に特化したPRレビューエージェント群 ファイル同期の安全規則(アトミック書き込み・バリデーション等)のような 妥協不可な要件のレビューに活用できる

プロジェクト運用系

プラグイン概要本プロジェクトでの用途
commit-commands commit/push/PR作成のgitワークフローを簡略化するコマンド タスク完了ごとのコミット運用(本セッションでも都度実施している)を定型化できる
claude-md-management CLAUDE.md の品質監査・セッション学習の記録・鮮度維持 本リポジトリの CLAUDE.md 自体の陳腐化を防ぐのに直接使える
hookify 会話パターンの分析や明示指示から Hook を自動生成 「settings/schedules.json を直接編集しようとしたら止める」といった ファイル同期の安全規則をHookとして機械的に強制する際の足がかりになる
security-guidance パターンベースの警告 + LLMによるdiffレビュー + コミットレビューで インジェクション・XSS・SSRF・ハードコードされた秘密情報等25種以上の脆弱性を検出 将来実装する /api/schedules Route Handler のセキュリティレビューに有用
skill-creator 新規Skillの作成・既存Skillの改善・性能計測 このプロジェクト独自のワークフロー(例: schedules.json編集手順)をSkill化したくなった時に使う

インストール方法(参考)

# 公式マーケットプレイスからインストールする一般形
/plugin install typescript-lsp@claude-plugins-official
/plugin install claude-md-management@claude-plugins-official

注意

  • code-review・code-simplifier といった類似プラグインも マーケットプレイスには存在するが、Claude Codeにはこれらに相当する Skill が 標準で組み込まれている場合がある。重複を避けるため、導入前に手元の環境で 既に同等の機能が使えないか確認すること
  • 本教材はカタログ化のみで、実際のインストール・apm.yml への追加は行っていない
