Catch-up · Agent Tool Management

Agent Tool 管理ツール(MCP・Skill 等)学習インデックス

MCP サーバーや Skill を発見・インストール・共有・管理するためのツールと、 npm/pip のように依存を宣言・ロック・再現する「Agent Package Manager」系のツールを あわせて8つ調査し、比較材料としてまとめました。 調査結果は docs/catch-up/agent-tool-management/research.md を元にしています。

この教材の使い方

本教材は特定のツールを推奨しません。8つの候補それぞれの特徴・メリット・デメリットを 並べているので、プロジェクトの状況(チーム規模、セキュリティ要件、対応クライアント、 ガバナンス要件等)に応じてご自身で判断する際の材料としてご利用ください。 前半(02〜05)は「発見・インストール」寄り、後半(06〜08)は「宣言的依存管理・再現性」寄りのツールで、 そもそも package manager 級のツールまで必要かを判断できるよう構成しています。 最後の「比較まとめ」ページに一覧表と判断手順があります。
